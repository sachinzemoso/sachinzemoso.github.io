About me

Here are key highlights of the projects undertaken.

HR portal

Synergy application streamlines and automates human resource processes, encompassing employee data management, leave and attendance tracking, recruitment, performance management, and more. It enhances efficiency, centralizes information, and promotes data-driven decision-making for HR professionals, ultimately fostering a more organized and responsive workforce management.

Contibutions

  1. Oversaw project development planning, crafted the architecture for the Liferay project, conducted code reviews, and provided solutions.
  2. Created a monitoring system utilizing Chart.js to track email activity, user navigation logins, schedulers, and interactions with Google Drive.
  3. Designed initial mock Bootstrap layouts, developed portlets, and added entertaining online games to the portal.

Loan settlement portal

Payars automates loan transaction management, ensuring precision and efficiency in repayments, interest calculations, and settlements. It offers automated notifications, secure transactions, and robust reconciliation, providing a transparent platform for borrowers and lenders to optimize financial processes.

Contibutions

  1. Oversaw project development planning, crafted the architecture for the Springboot project, conducted code reviews, and provided solutions.
  2. Established an AWS S3 bucket to store dynamically generated financial documents utilizing a PDF template.
  3. Designed a rule system based on Drools that generates distinct loan settlement plans for customers, allowing them to choose and make payments through an integrated payment gateway.

Parcel sorting application

Netsort, a Java Spring Boot-based standalone application, is a robust control system overseeing parcel sorting operations. It optimizes parcel processing through seamless integration with multiple physical hardware machines, offering centralized monitoring and streamlined sorting. Equipped with real-time communication and a user-friendly interface, it significantly enhances efficiency and reliability in logistics and distribution centers.

Contibutions

  1. Built a multi-threaded Spring Boot application that establishes connections with various hardware machines for message exchange using UDP.
  2. Created a Java sockets-based simulator to test the primary application instead of actual hardware machines.
  3. Created a Spring Boot-based web portal to showcase processed parcels, their current status, and additional information.
